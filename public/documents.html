API Documentation

API Routes Overview

Method Endpoint Description Middleware
POST /register Registers a new user.
POST /login Authenticates a user and returns a token.
POST /logout Logs out the authenticated user. auth:api
GET /welcome Displays the user dashboard. auth:api
GET /articles Lists all articles. auth:api, permission:view-post
GET /articles/{id} Displays a specific article. auth:api, permission:view-post
POST /articles Creates a new article. auth:api, permission:create-post
PUT /articles/{id} Updates an article. auth:api, permission:edit-post
DELETE /articles/{id} Deletes an article. auth:api, permission:delete-post
GET /users Lists all users (Admin only). auth:api, role:Admin
POST /users/{id}/assign-role Assigns a role to a user (Admin only). auth:api, role:Admin
POST /users/{id}/remove-role Removes a role from a user (Admin only). auth:api, role:Admin
POST /users/{id}/assign-permission Assigns a permission to a user (Admin only). auth:api, role:Admin
POST /users/{id}/remove-permission Removes a permission from a user (Admin only). auth:api, role:Admin
GET /roles Lists all roles (Admin only). auth:api, role:Admin
POST /roles Creates a new role (Admin only). auth:api, role:Admin
GET /permissions Lists all permissions (Admin only). auth:api, role:Admin
POST /permissions Creates a new permission (Admin only). auth:api, role:Admin

Authentication API

Register

URL: /register

Method: POST

Description: Register a new user.

Request Body:

{
    "name" : "string (required, max: 255)",
    "email" : "string (required, email, max: 255, unique: users)",
     "password": {
        "required",
        "string",
        "confirmed",
        "min: 8",
        "mixedCase",
        "numbers",
        "symbols",
        "uncompromised"
    }
}

Response:

{
    "token": "string (token)",
}

Login

URL: /login

Method: POST

Description: Login a user.

Request Body:

{
    "email": "string (required, email, max: 255)",
    "password": "string (required)"
}

Response:

{
    "token": "string (token)",
}

Logout

URL: /logout

Method: POST

Headers: Authorization: Bearer {token}

Description: Log the user out of the application.

Response:

{
    "message": "User logged out successfully"
}
